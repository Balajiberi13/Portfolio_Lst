# Portfolio Starter

A simple responsive portfolio starter project with HTML, CSS, and JavaScript.

## Files

- `index.html` — homepage layout and sections
- `styles.css` — responsive design and styling
- `script.js` — smooth scrolling and form handling

## Usage

1. Open `index.html` in your browser.
2. Customize text, links, and project cards.
3. Replace placeholder content with your own portfolio items.

## Tips

- Add your resume link to the header or contact section.
- Replace project cards with real screenshots and descriptions.
- Add more sections such as `Skills`, `Experience`, or `Testimonials`.
